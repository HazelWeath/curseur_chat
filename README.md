# Curseurs Personnalisés - Scottish Fold Pixel Art

Ce dépôt contient une collection de curseurs système personnalisés en pixel art (34x34) inspirés d'un chat écossais (*Scottish Fold*) orange et blanc. Vous y trouverez différentes déclinaisons de ses postures (l'espion caché derrière la barre, le chat qui dort, etc.).

---

## Guide d'installation sur Windows

Suivez ces instructions pour télécharger, installer et configurer proprement ces curseurs personnalisés sur votre système Windows (compatible Windows 10 et Windows 11).

### Étape 1 : Télécharger les fichiers depuis GitHub
1. En haut de cette page GitHub, cliquez sur le bouton vert **Code**.
2. Sélectionnez **Download ZIP** dans le menu déroulant.
3. Une fois le téléchargement terminé, localisez le fichier `.zip` dans votre dossier de téléchargements.
4. Faites un clic droit sur le fichier `.zip` et choisissez **Extraire tout...**, puis validez pour obtenir le dossier contenant les fichiers fonctionnels du curseur :
   * Les fichiers `.cur` (pour les curseurs fixes)
   * Les fichiers `.ani` (pour les curseurs animés)

### Étape 2 : Placer les fichiers dans le dossier système Windows
> **Important :** Pour éviter que Windows ne réinitialise vos curseurs personnalisés par défaut à chaque redémarrage de l'ordinateur, il est indispensable de les stocker dans le dossier système officiel de Windows.

1. Ouvrez l'**Explorateur de fichiers** de Windows.
2. Copiez le chemin d'accès suivant `C:\Windows\Cursors`, collez-le dans la barre d'adresse tout en haut, puis appuyez sur **Entrée**.
3. **Pour une organisation propre :** Faites un clic droit dans un espace vide de ce dossier, sélectionnez **Nouveau > Dossier**, et nommez-le `Curseurs_Chat`.
4. Ouvrez ce nouveau dossier `Curseurs_Chat` et copiez-y (ou glissez-y) les fichiers `.cur` ou `.ani` que vous avez téléchargés et extraits à l'étape 1.
5. Une fenêtre pop-up de sécurité Windows apparaîtra pour demander les droits d'administrateur pour modifier un dossier système. Cliquez simplement sur **Continuer**.

### Étape 3 : Activer les curseurs dans les paramètres de Windows
1. Ouvrez le menu Démarrer de Windows, tapez **Paramètres de la souris** et appuyez sur **Entrée**.
2. Dans la fenêtre des paramètres qui s'affiche :
   * **Sur Windows 10 :** Cliquez sur le lien **Options supplémentaires de la souris** (situé à droite ou tout en bas).
   * **Sur Windows 11 :** Faites défiler les options vers le bas et cliquez sur **Paramètres avancés de la souris**.
3. Une petite fenêtre classique intitulée *Propriétés de : Souris* va s'ouvrir. Cliquez sur l'onglet **Pointeurs** en haut.
4. Dans la liste centrale intitulée **Personnaliser**, faites défiler les éléments et cliquez sur le type de curseur que vous souhaitez modifier :
   * *Exemple : Pour changer votre curseur d'écriture, sélectionnez la ligne **Sélection de texte**.*
5. Cliquez sur le bouton **Parcourir...** en bas à droite.
6. La fenêtre s'ouvre directement dans le bon répertoire système. Double-cliquez sur votre dossier `Curseurs_Chat`.
7. Sélectionnez le fichier correspondant à la posture souhaitée (par exemple, le chat espion aigri pour l'écriture) et cliquez sur **Ouvrir**.
8. Pour valider, cliquez sur le bouton **Appliquer**, puis sur **OK**. Vos nouveaux curseurs sont prêts !
